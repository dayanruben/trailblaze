Usage: trailblaze session [-hV] [COMMAND]
Every blaze records a session — save it as a replayable trail
  -h, --help      Show this help message and exit.
  -V, --version   Print version information and exit.
Commands:
  start      Start a new session with automatic video and log capture
  stop       Stop the current session and finalize captures
  save       Write the recorded steps to a *.trail.yaml file you can replay
               later (does not end the session)
  recording  Output the recording YAML for a session
  info       Show information about a session
  list       List recent sessions
  artifacts  List artifacts in a session
  delete     Delete a session's logs and artifacts
  end        End the CLI session and release the device (deprecated: use 'stop'
               instead)
  report     Generate an HTML or JSON report for this session
