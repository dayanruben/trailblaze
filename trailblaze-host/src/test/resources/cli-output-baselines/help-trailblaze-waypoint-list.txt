Usage: trailblaze waypoint list [-hV] [--root=<root>]
List all waypoint definitions from active packs (workspace + framework
classpath)
and any additional *.waypoint.yaml files discovered under --root.
  -h, --help          Show this help message and exit.
      --root=<root>   Additional directory to scan for *.waypoint.yaml files
                        (default: ./trails, resolved against the current
                        working directory). Pack waypoints are always included
                        regardless of --root.
  -V, --version       Print version information and exit.
