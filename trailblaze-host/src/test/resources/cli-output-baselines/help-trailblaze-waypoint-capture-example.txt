Usage: trailblaze waypoint capture-example [-hV] [--force] --def=<defId>
       [--root=<root>] [--session=<session>] [--step=<step>]
       [<positionalLogFile>]
Capture a sibling <id>.example.json + screenshot next to the waypoint YAML.
Picks the raw (un-annotated) screenshot twin from the source session.
      [<positionalLogFile>] Path to a *_TrailblazeLlmRequestLog.json
                              (alternative to --session/--step)
      --def=<defId>         Waypoint id to capture an example for (required)
      --force               Overwrite an existing example pair without
                              prompting.
  -h, --help                Show this help message and exit.
      --root=<root>         Root directory to scan for *.waypoint.yaml files
                              (default: ./trails)
      --session=<session>   Session log directory (containing
                              *_TrailblazeLlmRequestLog.json files)
      --step=<step>         1-based step within the session (default: last step)
  -V, --version             Print version information and exit.
