Usage: trailblaze snapshot [-hvV] [--all] [--bounds] [--offscreen]
                           [--screenshot] -d=<device> [--headless=<headless>]
Capture the current screen's UI tree (fast, no AI, no actions)
      --all               Show all visible elements, including those normally
                            filtered as non-interactive
      --bounds            Include bounding box {x,y,w,h} for each element
  -d, --device=<device>   Device: platform (android, ios, web) or platform/id.
                            Required.
  -h, --help              Show this help message and exit.
      --headless=<headless>
                          For --device web/...: launch the Playwright browser
                            headless. When omitted, falls back to the persisted
                            `web-headless` config (see `trailblaze config
                            web-headless`). Pass --headless=false to force a
                            visible browser, --headless=true to force headless.
                            Ignored for non-web devices.
      --offscreen         Include offscreen elements marked (offscreen)
      --screenshot        Save a screenshot to disk and print the file path
  -v, --verbose           Enable verbose output
  -V, --version           Print version information and exit.
