Usage: trailblaze ask [-hvV] -d=<device> [--headless=<headless>]
                      <questionWords>...
Ask a question about what's on screen (uses AI vision, no actions taken)
      <questionWords>...   Question about the screen (e.g., 'What's the current
                             balance?')
  -d, --device=<device>    Device: platform (android, ios, web) or platform/id
                             (e.g., android/emulator-5554). Required.
  -h, --help               Show this help message and exit.
      --headless=<headless>
                           For --device web/...: launch the Playwright browser
                             headless. When omitted, falls back to the
                             persisted `web-headless` config (see `trailblaze
                             config web-headless`). Pass --headless=false to
                             force a visible browser, --headless=true to force
                             headless. Ignored for non-web devices.
  -v, --verbose            Enable verbose output (show daemon logs, MCP calls)
  -V, --version            Print version information and exit.
