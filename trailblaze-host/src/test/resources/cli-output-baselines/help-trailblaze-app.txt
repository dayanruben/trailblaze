Usage: trailblaze app [-hV] [--foreground] [--headless] [--status] [--stop]
Start or stop the Trailblaze daemon (background service that drives devices)
      --foreground   Run in foreground (blocks terminal). Use for debugging
                       with an attached IDE.
  -h, --help         Show this help message and exit.
      --headless     Start in headless mode (daemon only, no GUI)
      --status       Check if the daemon is running
      --stop         Stop the running daemon
  -V, --version      Print version information and exit.
