Usage: trailblaze desktop snapshot [-hV] [--json] [--out=<out>] [--port=<port>]
Capture a screen-state snapshot of the running Trailblaze desktop window via
the Compose RPC server.
  -h, --help          Show this help message and exit.
      --json          Print the full GetScreenStateResponse as JSON instead of
                        the human-readable summary.
      --out=<out>     When set, decode the response's base64 screenshot and
                        write the PNG to this path. Skips the file write if the
                        response has no screenshot.
      --port=<port>   Port the desktop app's Compose RPC server is listening on
                        (default: 52600)
  -V, --version       Print version information and exit.
