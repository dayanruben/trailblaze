Usage: trailblaze waypoint segment list [-hV] [--root=<root>]
                                        --session=<session>
List trail segments observed in a session log directory.
A segment is a transition from one matched waypoint to another, with the tool
calls that drove it.
  -h, --help                Show this help message and exit.
      --root=<root>         Additional directory to scan for *.waypoint.yaml
                              files (default: ./trails, resolved against the
                              current working directory). Pack waypoints are
                              always included regardless of --root.
      --session=<session>   Session log directory (containing *.json log files)
  -V, --version             Print version information and exit.
