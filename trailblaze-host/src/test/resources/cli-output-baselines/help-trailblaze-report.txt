Usage: trailblaze report [-hV] [--open] [--format=<format>] [--id=<id>]
Generate an HTML or JSON report from session recordings
      --format=<format>   Output format: html (default) or json — JSON emits a
                            CiSummaryReport artifact
  -h, --help              Show this help message and exit.
      --id=<id>           Session ID to report on (defaults to all sessions)
      --open              Open the report in the default browser after
                            generation (HTML only)
  -V, --version           Print version information and exit.
