Usage: trailblaze session save [-hV] [--id=<id>] [-t=<title>]
Write the recorded steps to a *.trail.yaml file you can replay later (does not
end the session)
  -h, --help            Show this help message and exit.
      --id=<id>         Session ID to save (defaults to current session,
                          supports prefix matching)
  -t, --title=<title>   Title for the saved trail (uses session title if not
                          specified)
  -V, --version         Print version information and exit.
