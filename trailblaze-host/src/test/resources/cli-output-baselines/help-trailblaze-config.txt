Usage: trailblaze config [-hV] [<key>] [<value>] [COMMAND]
View and set configuration (target app, device defaults, AI provider)
      [<key>]     Config key to get or set
      [<value>]   Value to set
  -h, --help      Show this help message and exit.
  -V, --version   Print version information and exit.
Commands:
  show    Show all settings and authentication status
  target  List or set the target app
  models  List available LLM models by provider
